### 注意事项
-----
#### 歌曲
播放器中的歌曲来自各大音乐平台，歌曲版权归各大平台享有。请支持正版音乐。
#### 维护
希望大家一起来维护这个迷你的音乐App，欢迎提交维护后的代码。
#### 其它
作者很忙，非诚勿扰。欢迎提交 BUG 及好的建议，不欢迎一上来就问一些基础的东西及唠东唠西。
